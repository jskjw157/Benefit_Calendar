# 혜택 캘린더 (BenefitCal) 로컬 개발 가이드

> 목적: MVP 로컬 개발 환경을 빠르게 구성하기 위한 최소 가이드.

## 1) 필수 환경
- Node.js 18 이상
- npm
- PostgreSQL 14 이상 (선택: 백엔드/DB 작업 시)

## 2) 프론트엔드 실행
```bash
npm install
npm run dev
```
- 기본 접속: `http://localhost:3000`

## 3) DB 초기화 (백엔드 작업 시)
```bash
# 스키마 적용
psql -f db/migrations/001_init.sql

# 샘플 데이터 적재
psql -f db/seeds/sample_data.sql
```

## 4) 환경 변수
- 현재 단계에서는 필수 환경 변수가 없습니다.
- 백엔드 연동 시점에 아래 항목을 추가 예정입니다.
  - `DATABASE_URL`
  - `JWT_SECRET`
  - `API_BASE_URL`
